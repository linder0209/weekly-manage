# weekly-manage
This is a webapp for weekly management.

## Install Build Server Test

### Install
At first wo should be installed nodejs(\>0.10.x).

```
npm install -g bower
npm install -g grunt
npm install
bower install
grunt server
```

### Unit Test
```
grunt test
```

### Build
```
grunt build
```
